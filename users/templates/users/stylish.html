Stay Flexy

And now! Behold! I demonstrate the Bootstrap flexy container grid system. We love these lil rows and columns, yes we do. Neat little style classes to make everything super easy.

Ugly border warning for the below (!!) Yes, I know, and I'm sorry, but ya girl has to check where things start and end, okay? And I find that visualising things tends to help me out (not just for code, but for learning anything, really). Lemme see dat padding and dem margins with my own human eyeballs!

So, some things to note for the below are:
  • We have a nice .container with two row elements:
    1. The first row for headers,
    2. The last row for text content.
  • Try and resize the window and check the responsive breakpoints here:
    1. The heading cols are gonna stack up with a smaller viewport.
    2. The text content cols will follow after all the headings, because they live in their own row.
  • align-items-center on a .row will y-axis centrally align the baby cols within in their row. Note how the borders show different heights of the cols.
  • Our baby heading cols have different styling:
    1. Heading One has no text alignment, so goes to the left by default.
    2. Heading Two has text-center alignment, so goes to the middle.
    3. Heading Three has text-end alignment, so goes to the right.
  • Our baby text content cols have different styling too, described below as part of the actual content.

class Cat:
  def __init__(self, name):
      self.name = name
      self.mood = "grumpy"

  def receive_food(self):
      self.mood = "pleased (for now)"
      print(f"{self.name}: *eats* ...I suppose you may live another day.")

  def receive_pet(self):
      if self.mood == "grumpy":
          print(f"{self.name}: *swats your hand* How dare you?!")
      else:
          print(f"{self.name}: *purrs* You have been deemed worthy.")

# Example usage
